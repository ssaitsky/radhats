# Hat Photos

Drop your hat photos in this folder named exactly:

```
hat-01.jpg
hat-02.jpg
...
hat-50.jpg
```

Each photo automatically replaces the placeholder for that hat number on the site. Square crops look best. No code changes needed, just add the files, commit, and push.

The hat numbers match the order in `hats.js` (No. 01 = The First One, No. 02 = Franti, etc.).
